✅ 功能说明
═══════════════════════════════════════════════════════════════

  ❌ 原问题：传统折叠式答案，交互性差

  ✅ 已解决：
     - 创建精美交互式练习题组件
     - 支持单选题和多选题
     - 圆形checkbox可点击选择
     - 选对显示绿色对勾 ✓
     - 选错显示红色叉叉 ✗
     - 正确答案自动标识
     - 即时反馈和详细解析


🎨 UI特点
═══════════════════════════════════════════════════════════════

  精美设计：
  ✓ 蓝紫渐变背景
  ✓ 圆形Q图标
  ✓ 字母标识（A/B/C/D）
  ✓ 圆形checkbox
  ✓ 渐变提交按钮

  状态样式：
  ○ 未选中 - 灰色空心圆
  ● 已选中 - 蓝色实心圆
  ✓ 正确   - 绿色对勾
  ✗ 错误   - 红色叉叉


🎯 交互流程
═══════════════════════════════════════════════════════════════

  1. 阅读问题
     ↓
  2. 点击选项（圆形checkbox）
     ↓
  3. 点击"提交答案"
     ↓
  4. 查看结果：
     - 正确：绿色✓ + "回答正确！🎉"
     - 错误：红色✗ + 显示正确答案
     ↓
  5. 查看详细解析
     ↓
  6. 点击"重新答题"继续练习


📝 使用方法
═══════════════════════════════════════════════════════════════

  在 Markdown 中添加：

  ```quiz-json
  {
    "question": "什么是机器学习？",
    "type": "single",
    "options": [
      {
        "id": "a",
        "text": "选项A",
        "isCorrect": false
      },
      {
        "id": "b",
        "text": "选项B",
        "isCorrect": true
      }
    ],
    "explanation": "答案解析..."
  }
  ```

  题型：
  - single   → 单选题
  - multiple → 多选题

🔧 技术实现
═══════════════════════════════════════════════════════════════

  组件架构：
  QuizParser (JSON解析)
      ↓
  QuizQuestion (交互界面)
      ↓
  MarkdownViewer (集成)

  状态管理：
  - selectedOptions（选中的选项）
  - showResult（是否显示结果）

  功能：
  - 选项点击处理
  - 答案验证逻辑
  - 结果展示
  - 重置功能
