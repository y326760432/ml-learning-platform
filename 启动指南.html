🚀 MLearning 启动指南

机器学习可视化教学平台

⚠️ 重要提示： 由于项目路径包含中文字符（非代码、机器学习网站），PowerShell 无法正确处理。请使用以下方法之一启动项目。

方法1：使用批处理文件 推荐

最简单的方式，无需任何命令行操作！

  1. 首次安装（只需一次）
    双击项目根目录的 install.bat 文件
  2. 启动开发服务器
    双击项目根目录的 start-dev.bat 文件
  3. 访问网站
    打开浏览器，访问 http://localhost:3000

方法2：使用命令提示符（CMD）

使用 CMD 而不是 PowerShell，对中文路径支持更好。

  1. 打开 CMD
    按 Win + R，输入 cmd，回车
  2. 进入项目目录
    d:
    cd cursor-demo\非代码\机器学习网站
  3. 启动开发服务器
    npm run dev
  4. 访问网站
    浏览器打开 http://localhost:3000

方法3：重命名文件夹

彻底解决中文路径问题。

  1. 创建新的英文路径文件夹，例如：
    D:\mllearning\
  2. 将所有项目文件复制到新文件夹
  3. 在新位置打开终端（CMD 或 PowerShell 都可以）
  4. 运行命令：
    npm install
    npm run dev
