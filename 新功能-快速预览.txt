  学什么？
  • 梯度下降是如何工作的
  • 学习率的重要性
  • 损失函数优化过程

  ─────────────────────────────────────

  玩什么？

  • 查看各簇点数统计
  
  学什么？
  • K-Means迭代过程

  • 如何选择K值

🎮 操作指南
═════════════════════════════════════════════════════

基础控制
  ▶️  播放  - 开始动画演示
  ⏸️  暂停  - 暂停查看细节
  🔄  重置  - 重新开始


参数调整
  📊 学习率     - 影响收敛速度
  🎯 聚类数K    - 改变簇的数量
  ⚡ 动画速度   - 0.5x / 1x / 1.5x / 2x


实时数据
  • 迭代次数
  • 当前参数
  • 损失值
  • 收敛状态


💡 使用建议
═════════════════════════════════════════════════════

📚 学习路径
  ┌────────────────────────────────────┐
  │ 第1步：线性回归（最简单）           │
  │   → 理解基础优化过程                │
  │                                    │
  │ 第2步：K-Means（中等）             │
  │   → 了解无监督学习                  │
  │                                    │

  │   → 掌握深度学习基础                │

🎯 实验建议
  1. 先完整观看一遍动画
  2. 暂停，仔细观察每个步骤
  3. 调整参数，对比不同效果
  4. 重复几次加深理解


⚡ 实验示例
  
  线性回归实验：
    • 学习率 0.001 → 收敛很慢
    • 学习率 0.01  → 正常收敛
    • 学习率 0.1   → 快速收敛（可能震荡）
  
  K-Means实验：
    • K=2  → 简单分组
    • K=3  → 符合数据分布
    • K=6  → 过度分割

❓ 常见问题
═════════════════════════════════════════════════════

Q: 动画太快看不清？
A: 调整速度为 0.5x，或随时暂停查看

Q: 参数改不了？
A: 需要先暂停动画才能调整参数

Q: 想重新看一遍？
A: 点击重置按钮即可

Q: 支持手机吗？
A: 支持！但建议用电脑获得最佳体验
