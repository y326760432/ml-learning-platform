  🚀 Cloudflare Pages 快速部署指南
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✨ 最简单的方法：

    1. 双击运行：deploy-cloudflare.bat
    2. 等待部署完成
    3. 访问提供的网址

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 手动部署步骤：

步骤 1: 登录 Cloudflare（首次部署需要）
    npx wrangler login

步骤 2: 构建项目
    npm run build

步骤 3: 部署
    npx wrangler pages deploy out --project-name=ml-learning-platform

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 重要说明：

1. 不要使用 "npx wrangler deploy"（那是 Workers 用的）
2. 要使用 "npx wrangler pages deploy"（这是 Pages 用的）
3. 确保已经运行过 "npm run build" 生成 out 目录

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 部署成功后：

访问：https://ml-learning-platform.pages.dev
（或 Cloudflare 提供的其他网址）
